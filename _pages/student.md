招生说明
======
* 需要这样的学生（<u>排序有先后</u>）
 + 爱党爱国，热爱人民；
 + 有自驱力、数学好、语文好、编程好、英语好（排名有先后）；
 + 自动化类、数学类、计算机类专业毕业；
 + 第一学历二本以上；
 + 有科研经历者优先（各级各类型创新创业大赛、电子设计大赛、数学建模大赛等不属于认可的科研经历）；
 + 有科研成果者优先（中科院一二区论文、中科院三四区非OA论文、中文EI论文、CCF-A/B/C认定的会议论文、已转化的授权专利等）；
 + 低年级本科生同样欢迎。
* 承诺
 + 学术思想自由；
 + 科研时间自由；
 + 聚焦学术前沿；
 + 亲自指导；
 + 不做横向。
* 不承诺一直有充足的科研经费（项目基金不可控、个人能力有限）
* 有意者请将个人简历发到邮箱wenjie@nuc.edu.cn
